## 1. 本书建模口径

书中燃料电池系统模型由电堆模型和外部辅助部件模型组成：

```text
系统模型 = 电堆模型 + 空气压缩机 + 进气/排气管路 + 中冷器 + 加湿器 + 阀/背压边界 + 控制器
```

电堆模型进一步拆成：

```text
电堆模型 = 阴极流量模型 + 阳极流量模型 + 膜水合模型 + 电堆电压模型
```

主要假设包括：

- 气体按理想气体处理。
- 阴极和阳极控制体内气体组分空间均匀。
- 堆温度由冷却系统维持在给定温度，示例中常取 80 degC。
- 进入流道的气体温度、湿度、压力和组分可作为集总边界。
- 液态水体积对气腔体积的影响忽略；液态水通过饱和裁剪和出流项处理。

对当前 cEGR 项目的直接含义：

- 电堆入口状态必须由混合后的 `O2/N2/H2O_v/liquid/T/p/RH` 计算，不能只用新鲜空气倍率代替。
- 阴极和阳极至少应有气体库存质量状态，分压由库存质量、温度和体积反算。
- Faraday 反应消耗、产水、膜传水和液水排出应作为质量守恒源项出现。
- BOP 不宜全部等效为常数边界；空气机、管路、加湿器和排气管应保留低维动态或可追溯代数关系。

## 2. 阴极流量模型

### 2.1 状态量和控制方程

阴极控制体状态采用氧气、氮气和水蒸气质量：

```text
d(m_O2,ca)/dt = W_O2,ca,in - W_O2,ca,out - W_O2,reacted
d(m_N2,ca)/dt = W_N2,ca,in - W_N2,ca,out
d(m_v,ca)/dt  = W_v,ca,in - W_v,ca,out + W_v,ca,gen + W_v,membr - W_l,ca,out
```

符号含义：

- `m_O2,ca`、`m_N2,ca`、`m_v,ca`：阴极氧气、氮气、水蒸气库存质量。
- `W_i,ca,in`：进入阴极的组分质量流量。
- `W_i,ca,out`：离开阴极的组分质量流量。
- `W_O2,reacted`：电化学反应消耗的氧气。
- `W_v,ca,gen`：阴极反应生成水。
- `W_v,membr`：膜水传输进入或离开阴极侧的水蒸气等效源项。
- `W_l,ca,out`：离开阴极的液态水质量流量。

这组方程可作为当前 `PEMFCStack` 阴极控制体的主方程骨架。对 cEGR，入口 `W_O2/N2/v,ca,in` 应由新鲜空气、加湿器干侧出口和回流尾气混合后计算，而不是由固定空气摩尔分数直接给定。

### 2.2 入口湿空气和含湿量

入口湿空气中氧气、氮气和水蒸气质量流量可按干气组分和含湿量拆分：

```text
W_O2,ca,in = x_O2,ca,in * W_a,ca,in
W_N2,ca,in = x_N2,ca,in * W_a,ca,in
W_v,ca,in  = W_ca,in - W_a,ca,in
```

干空气质量分数由摩尔分数和分子量换算：

```text
x_O2 = y_O2*M_O2 / (y_O2*M_O2 + (1-y_O2)*M_N2)
x_N2 = (1-y_O2)*M_N2 / (y_O2*M_O2 + (1-y_O2)*M_N2)
```

书中示例取空气中氧气摩尔分数 `y_O2 = 0.21`。cEGR 模型中，进入电堆前的有效 `y_O2,ca,in` 必须按混合气实际组成计算，不能固定为 0.21。

含湿量使用水蒸气分压和饱和蒸气压计算：

```text
omega = (M_v/M_dry) * phi*p_sat(T) / (p - phi*p_sat(T))
phi   = p_v / p_sat(T)
```

其中 `phi` 为相对湿度。图片中给出一个水饱和蒸气压经验式，温度单位和压力单位在写入代码前需与原书逐式核对。工程实现时建议沿用当前项目已统一的 `psatWaterKPa` 或等价函数，避免同一模型内出现多套饱和压单位。

### 2.3 阴极分压和总压

阴极各组分分压按理想气体计算：

```text
p_O2,ca = m_O2,ca * R_O2 * T_st / V_ca
p_N2,ca = m_N2,ca * R_N2 * T_st / V_ca
p_v,ca  = m_v,ca  * R_v  * T_st / V_ca
p_ca    = p_O2,ca + p_N2,ca + p_v,ca
```

这组关系对当前模型很关键：电压模型中的 `p_O2` 应来自阴极库存反算的代表分压，而不是直接来自入口边界或空气倍率命令。

### 2.4 阴极出口流量和组分分配

书中采用线性喷嘴/节流关系近似阴极出口总质量流量：

```text
W_ca,out = k_ca,out * (p_ca - p_rm)
```

其中 `p_rm` 是下游排气管或歧管压力。由于书中说明阴极出口压差较小，使用线性喷嘴方程计算；如果当前 cEGR 模型进入大压差或节流临界状态，应升级为可压缩节流方程。

出口干气组分和含湿量由当前控制体分压决定：

```text
y_O2,ca = p_O2,ca / p_ca
omega_ca,out = f(p_v,ca, p_ca, T_st)
W_a,ca,out = W_ca,out / (1 + omega_ca,out)
W_v,ca,out = W_ca,out - W_a,ca,out
W_O2,ca,out = x_O2,ca * W_a,ca,out
W_N2,ca,out = x_N2,ca * W_a,ca,out
```

工程落地建议：

- 出流组分优先按“库存质量分数/摩尔分数”分配，避免出口组分与控制体状态不一致。
- 先计算反应源项和膜传水源项，再做饱和裁剪和出口分配，可减少水蒸气库存被出流项抽成负值的风险。

### 2.5 阴极反应消耗、产水和液水裁剪

Faraday 源项：

```text
W_O2,reacted = M_O2 * n * I_st / (4F)
W_v,ca,gen   = M_v  * n * I_st / (2F)
```

其中 `n` 为单电池片数，`I_st` 为电堆电流，`F` 为法拉第常数。若当前模型使用电流密度 `i`，需要明确 `I_st = i*A_cell`，并区分单片电流和总堆电流。

液水处理采用饱和裁剪逻辑：

```text
m_v,max,ca = p_sat(T_st) * V_ca / (R_v*T_st)

if m_v,ca <= m_v,max,ca:
    m_l,ca = 0
else:
    m_l,ca = m_v,ca - m_v,max,ca
    m_v,ca = m_v,max,ca
```

这适合作为系统级低维模型的一阶近似。它不能预测局部液水淹没、流道液滴滞留或 GDL 饱和度分布，只能保证控制体水蒸气不超过饱和容量。

## 3. 阳极流量模型

### 3.1 状态量和控制方程

阳极控制体重点跟踪氢气和水蒸气：

```text
d(m_H2,an)/dt = W_H2,an,in - W_H2,an,out - W_H2,reacted
d(m_v,an)/dt  = W_v,an,in  - W_v,an,out  - W_v,membr
```

如果考虑阳极排水或液水滞留，可加入与阴极类似的液态水裁剪和液水出流项。当前 10 kW 主线采用开放式固定计量比供氢和固定背压边界时，仍建议保留 `m_H2,an` 和 `m_v,an` 库存，以保证 `p_H2` 和膜水边界有明确来源。

### 3.2 阳极入口和出口流量

阳极入口含湿氢气按含湿量拆分：

```text
W_H2,an,in = W_an,in / (1 + omega_an,in)
W_v,an,in  = W_an,in - W_H2,an,in
omega_an,in = (M_v/M_H2) * phi_an,in*p_sat(T_an,in) / (p_an,in - phi_an,in*p_sat(T_an,in))
```

阳极库存分压：

```text
p_H2,an = m_H2,an * R_H2 * T_st / V_an
p_v,an  = m_v,an  * R_v  * T_st / V_an
p_an    = p_H2,an + p_v,an
```

出口流量结构与阴极类似，可由阳极出口压力和下游压力差决定，再按当前控制体组分分配：

```text
omega_an,out = (M_v/M_H2) * p_v,an / (p_an - p_v,an)
W_H2,an,out = W_an,out / (1 + omega_an,out)
W_v,an,out  = W_an,out - W_H2,an,out
```

阳极反应消耗：

```text
W_H2,reacted = M_H2 * n * I_st / (2F)
```

### 3.3 对当前模型的取舍

当前项目不是阳极循环专题，因此不建议展开复杂阳极吹扫控制。但以下三点不应省略：

- `p_H2,an` 必须从阳极库存或明确边界反算，作为 Nernst 电压输入。
- `W_H2,reacted` 必须按 Faraday 定律进入氢气守恒。
- 阳极水蒸气状态必须参与膜水合边界，否则膜水模型会只受阴极湿度驱动，物理上不闭合。

## 4. 膜水合模型

### 4.1 跨膜水通量

书中膜水传输包含两个过程：

```text
N_osmotic = n_d * i / F
N_diff    = -D_w * dc_v/dy
N_membr   = n_d*i/F - D_w*(c_v,ca - c_v,an)/L_m
```

其中：

- `N_osmotic`：电渗拖曳水通量。
- `N_diff`：浓度梯度导致的反扩散水通量。
- `N_membr`：跨膜净水通量。
- `i`：电流密度。
- `L_m`：膜厚。
- `c_v,ca`、`c_v,an`：阴极和阳极侧水浓度。

书中公式的正方向需在代码中固定。对当前项目建议统一定义：

```text
N_membr > 0 表示水从阳极传到阴极
```

若采用相反方向，所有阴极/阳极水源项必须同步反号，不能只改局部变量名。

跨膜水质量流量：

```text
W_v,membr = N_membr * M_v * A_fc * n
```

### 4.2 电渗拖曳系数、水活度和膜含水量

水活度：

```text
a_an = p_v,an / p_sat(T_st)
a_ca = p_v,ca / p_sat(T_st)
a_m  = (a_an + a_ca) / 2
```

膜含水量经验关系：

```text
lambda_m = 0.043 + 17.81*a_m - 39.85*a_m^2 + 36.0*a_m^3,  0 < a_m <= 1
lambda_m = 14 + 1.4*(a_m - 1),                             1 < a_m <= 3
```

电渗拖曳系数：

```text
n_d = 0.0029*lambda_m^2 + 0.05*lambda_m - 3.4e-19
```

水扩散系数采用温度修正和含水量分段函数：

```text
D_w = D_lambda * exp[2416*(1/303 - 1/T_st)]
```

书中给出的 `D_lambda` 为按 `lambda_m` 分段的经验式。后续实现时需要核对单位，尤其是 `cm^2/s` 与 `m^2/s` 的换算。

### 4.3 对当前 cEGR 模型的约束

膜水模型是新书里最适合直接强化当前模型的部分之一：

- `lambda_m` 应由阴阳极水活度共同决定，不宜只用阴极 RH。
- `eta_ohm` 应通过膜含水量或膜电阻反馈到电堆电压。
- `W_v,membr` 应成对进入阴极和阳极水蒸气守恒；一个侧为源项，另一侧必须为等量汇项。
- cEGR 提高入口湿度后，影响链路应是 `回流湿气 -> 阴极 p_v/RH -> lambda_m -> R_m/eta_ohm -> V_cell`，而不是直接给电压加修正系数。

## 5. 电堆电压模型

### 5.1 电压分解

书中采用常见半经验结构：

```text
V_cell = E - V_act - V_ohm - V_conc
V_stack = n * V_cell
```

这与当前项目记忆和现有主线模型的结构一致：

```text
V_cell = E_rev - eta_act - eta_ohm - eta_con
```

后续仍应保留分项输出，至少包括：

- `E_rev`
- `eta_act`
- `eta_ohm`
- `eta_con`
- `V_cell`
- `V_stack`

### 5.2 可逆电压

书中可逆电压采用 Nernst 型表达，核心依赖为：

```text
E = f(T_st, p_H2, p_O2)
```

常用结构为：

```text
E = 1.229 - k_T*(T_st - 298.15)
    + (R*T_st/(2F))*ln(p_H2 * sqrt(p_O2))
```

注意事项：

- `p_H2` 应来自阳极代表分压。
- `p_O2` 应来自阴极代表分压。
- 若使用绝对压力，需归一化到参考压力或保持与标定参数一致。

### 5.3 活化、欧姆和浓差损失

活化损失使用 Amphlett 类半经验表达，结构上依赖温度、氧浓度和电流：

```text
V_act = xi_1 + xi_2*T_st + xi_3*T_st*ln(C_O2) + xi_4*T_st*ln(I_st)
C_O2 = f(p_O2, T_st)
```

欧姆损失：

```text
V_ohm = I_st * (R_m + R_c)
R_m   = rho_m * L_m / A_fc
rho_m = f(i, T_st, lambda_m)
```

浓差损失：

```text
V_conc = m * exp(n_c * I_st)
```

采用建议：

- 当前 10 kW 模型可继续使用已标定的极化参数，但应把 `lambda_m` 对 `R_m/eta_ohm` 的影响补强为可开关或可标定项。
- 活化和浓差项若继续使用拟合参数，应明确其适用范围来自无 EGR 极化曲线；cEGR 工况下只能通过 `p_O2/RH/T` 的物理链路外推，不能把 EGR 比例直接写入极化拟合式。
- 低氧或高湿工况下的浓差项需要设置限幅和诊断，避免指数项导致数值发散却没有物理解释。

## 6. 辅助部件模型

### 6.1 空气压缩机

书中将空压机作为电化学反应供氧的主要 BOP，强调低体积、低噪声、高速旋转和动态气流响应。压缩机特性可由 MAP 或多项式拟合：

```text
W_cp = B00 + B10*p_r + B20*p_r^2 + B01*omega_cp
       + B11*p_r*omega_cp + B02*omega_cp^2
```

压缩机出口温度使用等熵压缩和效率修正：

```text
T_cp,out = T_atm + T_atm/eta_cp * [(p_out/p_atm)^((gamma-1)/gamma) - 1]
```

对当前项目：

- 如果已有 DQ60 MAP，应优先使用真实或数字化 MAP；书中多项式只适合作为无 MAP 时的近似。
- cEGR 回流进入空压机前混合时，空压机入口气体不再是标准干空气，需在密度、含湿量和氧浓度计算中体现。
- 功耗、压比和喘振边界应作为诊断输出，不宜在未标定前作为最终效率结论。

### 6.2 进气管路

进气管路存在压力和流量损失。书中采用控制体气体动态：

```text
W_rm,out = k_rm,out * (p_rm - p_0)
dm_rm/dt = W_cp - W_rm,out
dp_rm/dt = (R_a/V_rm) * (W_cp*T_cp,out - W_rm,out*T_rm)
```

其中 `p_rm` 为进气管压力，`V_rm` 为管路体积。该结构适合当前模型从“固定压力边界”升级为“管腔库存 + 压力动态”。

### 6.3 中冷器

中冷器假设气体无质量损失，主要改变出口温度和相对湿度。书中默认中冷后适宜温度约为 80 degC，温度变化引起气体相对湿度变化。

当前项目采用建议：

- 若没有中冷器 UA 或性能曲线，可先用恒温边界或 `epsilon-NTU` 低维模型。
- 降温导致的过饱和必须触发冷凝裁剪，否则后续加湿器和电堆入口 RH 会被高估。
- 中冷器输出至少要给出 `T_out`、`p_out`、`W_v,out`、`liquid_condensed`。

### 6.4 加湿器

书中强调外接加湿器的作用是提高进堆空气含水量，以满足膜水合需求；但若相对湿度较低，膜可能干裂，影响安全和寿命。

加湿器入口干空气可按总压和水蒸气分压拆分：

```text
p_cl = p_da + p_v,cl
omega_cl = (M_v/M_a) * p_v,cl / p_da
W_a,cl = W_cl / (1 + omega_cl)
W_v,cl = W_cl - W_a,cl
```

加湿器出口：

```text
W_hm = W_cl + W_v,inj
p_hm = p_da + p_v,hm
phi_hm = p_v,hm / p_sat(T_hm)
```

对 GZS60/cEGR 的含义：

- 书中加湿器模型可作为“外部加湿器”基础，但本项目的 GZS60 是干湿两侧换湿器，不应简单写成外部注水 `W_v,inj`。
- 当前 GZS60 模型更应使用干湿两侧控制体或 Lumped-Mass 换湿模型。
- cEGR 回流尾气既是阴极入口水源，也是 GZS 湿侧水源，必须在水量账上闭合。

### 6.5 排气管路

书中排气管包含共压力阀，出口气体温度近似等于电堆出口空气温度。排气管压力动态：

```text
dp_rm/dt = (R*T_rm/V_rm) * (W_ca,out - W_rm,out)
```

排气出口流量采用可压缩节流关系，按 `p_atm/p_rm` 是否达到临界压比分段：

```text
W_rm,out = f(C_D, A_T, p_rm, T_rm, p_atm/p_rm, gamma)
```

对当前模型：

- 背压阀和排气阀不能长期只用固定压降代替。
- cEGR 分流点前后的压力会影响回流质量流量，阀模型需要至少保留流量系数、开度、上游压力、下游压力和临界压比。
- 若暂缺阀曲线，应明确标为工程默认，不用于定量结论。

## 7. 参数辨识方法

书中第 4.2 节说明复杂系统模型参数常分为系统梯度下降类和启发式优化类；因容易陷入局部最优，书中介绍了混合智能优化，例如蚱蜢优化算法、模拟退火以及 SA-GOA 混合算法。

对当前项目的实用价值：

- 电堆电压参数可继续采用极化曲线标定，但要保留分项约束，避免所有误差都被一个经验参数吸收。
- BOP 参数如 `k_out`、管腔体积、阀流量系数、加湿器传水系数、冷凝效率等，应按模块分层辨识，不建议一次性全局黑箱拟合。
- 参数辨识目标函数应同时包含电压误差、流量/压力误差、湿度误差和守恒残差，不应只拟合电压。
- 对 cEGR 场景，至少应使用无 EGR 基准、低负荷有 EGR、不同回流比例三个数据组交叉验证。

建议的辨识顺序：

```text
1. 无 EGR 电堆极化参数：E_rev / eta_act / eta_ohm / eta_con
2. 阴阳极控制体：k_in/k_out、有效体积、饱和裁剪参数
3. GZS60 加湿器：传水有效度、动态时间常数、压降系数
4. 空气机/阀/排气管：MAP、流量系数、背压边界
5. cEGR 分流：回流比、排气比、混合后 O2/RH/压力闭合
```

## 8. 与旧书资料的关系

两本书对当前项目的互补关系可以这样处理：

| 方面 | 《质子交换膜燃料电池系统及其控制》 | 《氢燃料电池多物理过程建模与仿真》 |
|---|---|---|
| 适用层级 | 系统控制模型、集总动态、BOP 方程 | 多物理过程、局部机理、源项细节 |
| 电堆状态 | 阴/阳极库存质量 + 膜水 + 电压 | 可展开到 CH/GDL/CL/PEM 多层 |
| BOP | 空压机、管路、中冷器、加湿器、排气管更直接 | BOP 范式较完整，但控制模型细节不如本书直接 |
| 当前项目优先级 | 作为系统模型控制方程和源项主参考 | 作为机理补强和复杂源项核查参考 |

因此，当前 10 kW cEGR 主线建议采用：

- 系统模型框架、阴/阳极流量、膜水合、电压、BOP 动态优先按本书整理。
- 液水、GDL/CL、膜内更细源项机理仍参考旧书和基础论文。
- 若两者公式有差异，先判断模型层级是否不同，不直接判定谁对谁错。

## 9. 对当前 10 kW GZS60 cEGR 模型的采用建议

### 9.1 可直接采用或优先对齐

- 阴极三状态守恒：`O2/N2/H2O_v`。
- 阳极两状态守恒：`H2/H2O_v`，必要时加液水状态。
- 分压由库存质量、温度和体积反算。
- Faraday 氧耗、氢耗和产水源项。
- 膜水合：`a_an/a_ca -> lambda_m -> n_d/D_w -> W_v,membr`。
- 电压分解：`V_cell = E - V_act - V_ohm - V_conc`。
- 空压机出口温度的等熵效率修正。
- 管路和排气管的压力动态控制体。

### 9.2 作为二级升级

- 阴极和阳极出口由线性喷嘴升级为可压缩节流。
- 中冷器从恒温边界升级为 UA 或有效度模型。
- GZS60 从有效度代数模型升级为干湿侧动态 Lumped-Mass 模型。
- 背压阀、EGR 阀和排气阀从固定压降升级为开度-流量-压差模型。
- 参数辨识从单目标极化拟合升级为多目标物理约束拟合。

### 9.3 暂不建议直接纳入主线

- 复杂启发式优化算法不应在方程尚未闭合时优先使用。
- 书中示例空压机多项式不应替代已有 DQ60 MAP。
- 未校核单位的膜水扩散系数、饱和蒸气压经验式不应直接写入代码。
- 液水局部分布、GDL 饱和度和 CL 传质不能由当前集总模型直接宣称。

## 10. 需要后续核对的具体点

1. 饱和蒸气压经验式的温度单位和压力单位。
2. 膜水扩散系数 `D_w` 的单位，尤其是 `cm^2/s` 与 `m^2/s`。
3. `N_membr` 正方向和阴/阳极水源项符号。
4. Nernst 式中 `p_H2`、`p_O2` 是否按 Pa、atm 或归一化压力输入。
5. 欧姆损失中膜电阻率 `rho_m` 经验式的电流密度单位。
6. 管路和排气管 `rm` 下标在原书中是否分别代表进气管、排气管或 return manifold。
7. 加湿器公式中 `W_v,inj` 与本项目 GZS60 膜换湿结构的等效关系。
8. 参数辨识章节是否还有未拍摄的目标函数定义和实验校验指标。
