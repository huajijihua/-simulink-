# 参考库分类与建模支撑索引 v01

本文档用于把 `00_支撑材料/参考建模材料` 中的文献、模型案例和项目标准整理成可用于 PEMFC-cEGR Simulink 建模核查的依据索引。用途不是替代逐篇精读，而是先明确每份资料能支撑什么、不能支撑什么，便于后续建立“方程-源项-参数-文献依据”的追溯链。

## 1. 使用原则

后续核查每一个正式方程时，按以下优先级寻找依据：

1. 基础物理化学公式优先查 `01_基础原理公式`：Nernst、Faraday、极化损失、膜水含量、电渗拖拽、反扩散、膜电导率、气体跨膜传质等。
2. 系统级集总模型优先查 `02_PEMFC系统级建模与仿真` 和 `06_书籍摘录与项目建模标准`：电堆动态、气-水-热耦合、系统控制接口、低维建模假设。
3. cEGR 相关机理和控制边界优先查 `03_cEGR阴极循环技术研究`：循环比、自加湿、入口氧浓度、低负荷高电位、氧饥饿、怠速和冷启动。
4. BOP 组件和控制模型优先查 `04_BOP组件与控制`：加湿器、空气供给、氧过量系数、阴极压力、MPC/鲁棒控制。
5. Simulink/Simscape 实现方式优先查 `05_成熟模型案例`：只作为结构、变量定义、组件实现和工程实现参考；不能单独作为理论公式出处。
6. 实验数据、设备说明书、标定曲线应作为参数和边界依据，不替代理论依据。

## 2. 支撑等级

| 等级 | 含义 | 可用于 |
|---|---|---|
| A | 可直接支撑公式、源项或成熟低维方程 | 正式模型方程引用、公式核查 |
| B | 支撑机理、趋势、建模假设或控制边界，公式需要结合项目适配 | 模型选择、简化说明、风险边界 |
| C | 成熟案例或实现参考，但不是原始理论来源 | Simulink/Simscape 架构、接口、变量命名、工程实现 |
| D | 背景、诊断或风险提示，不能单独支撑定量模型 | 约束说明、后续验证需求 |

## 5. 当前参考库能支撑的结论边界

可以较有把握支撑：

- 电堆电压分解：`V_cell = E_rev - eta_act - eta_ohm - eta_conc`。
- Nernst 电压、Faraday 氢耗/氧耗/产水、极化损失、膜水含量和膜电导率的基础表达。
- 阴/阳极气体和水的守恒框架。
- cEGR 通过改变阴极入口氧浓度、湿度和循环流量影响低负荷电压、自加湿和辅助功耗。
- 氧过量系数、阴极压力和空气供给动态控制是 PEMFC 系统级控制的核心对象。
- Simscape/AlphaPEM/两相开源模型可作为实现和验证流程参考。

目前不能直接作为最终定量结论的部分：

- 用近似空气机或循环泵代替真实设备 MAP 得出的功耗、压力和流量边界。
- 阀、分流器、分水器、冷凝器、换热器缺少实物参数时的定量结果。
- 集总膜水模型对局部干涸、液水淹没、GDL/CL 分布效应的预测。
- 缺少本项目 cEGR 台架验证时，对循环比最优值、低负荷耐久收益、怠速/冷启动节能量的定量外推。
