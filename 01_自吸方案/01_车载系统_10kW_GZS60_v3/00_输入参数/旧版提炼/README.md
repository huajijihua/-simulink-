# 旧版提炼输入资料说明

更新时间：2026-06-03

本目录只保存从初版 `01_车载系统_10kW_GZS60` 和 v2 `01_车载系统_10kW_GZS60_v2` 中提炼出的少量可复用输入资料。它不是 v3 当前主参数目录，也不代表这些参数已完成定量标定。

## 1. 使用原则

- v3 当前权威模型、脚本、验证结果仍以 `01_模型`、`02_脚本`、`03_说明`、`04_验证结果` 为准。
- 本目录资料只作为后续增强 v3 时的参考入口，不能直接覆盖 v3 当前参数。
- 初版和 v2 的旧模型、旧脚本、旧 `slprj`、旧 `.slxc`、历史 sweep/审计结果没有迁入。
- 若某份旧版资料要正式进入 v3，应先在 `03_说明/v3_模块方程与源项台账.md` 中登记方程、参数来源、支撑等级和待标定缺口。

## 2. 已保留资料

| 子目录 | 来源 | 内容 | 后续用途 |
|---|---|---|---|
| `空气机_DQ60_MAP数字化` | v2 输入参数 | DQ60 压升、功耗和坐标轴数字化 CSV | 后续接入 DQ60 MAP 或检查 MAP 内/外工况 |
| `台架数据接口` | 初版输入参数 | 标准节点信号 schema、台架数据模板、必需信号清单 | 后续对接 cEGR 台架数据和结果接口 |
| `BOP参数来源` | 初版输入参数 | BOP 参数包、GZS60 组件参数来源表 | 后续梳理阀、分水器、中冷器、GZS60 参数依据 |
| `GZS通用化数据与参数反演` | 初版结果 | GZS 系列标准四口数据、GZS80P 水量审计、Lumped-Mass 参数包和回放摘要 | 后续增强 GZS60 加湿器模型和可信区间 |
| `边界复核` | 初版输入参数 | 低电流台架边界复核表 | 后续复核 0.1 A/cm2 低负荷边界 |
| `v2参数治理参考` | v2 输入参数 | v2 默认参数表和输入清单 | 后续参考参数集中治理方式，不作为 v3 主参数 |

## 4. 删除旧目录后的边界

删除初版和 v2 目录后，v3 仍保留以下能力：

- 查到 DQ60 数字化 MAP 的处理后 CSV。
- 查到台架标准信号接口。
- 查到 BOP/GZS 参数来源和 GZS 通用化参数反演结果。
- 查到 v2 的关键建模纪律、守恒更新方式和验证教训。

删除旧目录后，以下内容不再保留：

- 旧模型可运行性。
- 旧脚本复跑能力。
- 初版历史图件、PPT、sweep 和过程审计文件。
- v2 历史三份验证结果 CSV/MD 的原始文件。

这个取舍是有意的：后续只在 v3 上继续建模研究，旧版只留下能支撑 v3 改进的参数、接口和经验。
