## 2. 从文献到当前模型的取舍

`1.pdf` 的基本形式是：

```text
m_cur = weighted_spectrum(i(t))
m_vol = weighted_histogram(V_cell(t))
phi = m_cur * m_vol
v_deg_rate = a * phi + b
T_life = allowable_voltage_loss / v_deg_rate
```

这套方法适合车辆动态负载谱。但当前项目主要是稳态/准稳态 EGR 工况扫描，暂时没有完整行驶循环。因此第一版把 `phi` 拆成几个可解释、可标定的等效因子：

```text
v_deg_rate = base_decay_mV_h * f_V * f_RH * f_T * f_j * f_cycle
```

其中：

- `f_V` 对应文献中的电压直方图风险权重，重点惩罚 `V_cell > 0.8 V`。
- `f_RH` 对应膜干燥风险和 cEGR 自加湿收益。
- `f_T` 是长时老化常用 Arrhenius 温度加速因子。
- `f_j` 保留低电流/高电流工况风险。
- `f_cycle` 是动态循环项，稳态扫描中取 1；后续有时间序列后可升级回 `m_cur * m_vol`。

## 3. 当前使用的公式

### 3.1 电压因子

```text
f_base_V = exp(k_potential_exp * (V_cell - V_ref))
over_high = max((V_cell - V_high) / V_scale_high, 0)
under_low = max((V_low - V_cell) / V_scale_low, 0)

f_V = sat(f_base_V * (1 + k_over_high * over_high^2)
          + k_low_voltage * under_low^2)
```

默认参数：

```text
V_ref = 0.75 V
V_high = 0.80 V
V_low = 0.60 V
```

依据：`1.pdf` 推荐较合适的电压区间约为 `0.6-0.8 V`；`3.pdf`、`5.pdf` 均把低负荷高电位限制作为耐久性策略。

### 3.2 湿度因子

```text
dryness = max((RH_min - RH_ca_in) / RH_min, 0)
f_RH = sat(1 + k_dry * dryness^2)
```

默认参数：

```text
RH_min = 0.70
RH_ref = 0.80
```

依据：`3.pdf`、`5.pdf` 都指出阴极循环带来入口湿度提升；第一版只把低湿作为干燥风险惩罚，不把过湿收益额外放大。

### 3.3 温度因子

```text
T_stack_K = T_stack_C + 273.15
f_T = exp(Ea / R * (1 / T_ref - 1 / T_stack_K))
```

默认参数：

```text
T_ref = 333.15 K
Ea = 25000 J/mol
```

依据：老化反应常用 Arrhenius 型温度加速；此处是等效参数，后续需要耐久数据标定。

### 3.4 电流和循环因子

```text
low_current = max((j_low - j) / j_low, 0)
high_current = max((j - j_high) / j_high, 0)
f_j = sat(1 + k_j_low * low_current^2 + k_j_high * high_current^2)

f_cycle = sat(1 + k_djdt * abs(dj/dt) / djdt_ref)
```

当前稳态扫描中 `f_cycle = 1`。

### 3.5 总衰减率、寿命和 ECSA 代理量

```text
life_damage_rate_mV_h = base_decay_mV_h * f_V * f_RH * f_T * f_j * f_cycle

delta_V_deg_mV = life_damage_rate_mV_h * duration_h

damage_index = delta_V_deg_mV / allowable_decay_mV

projected_life_to_EOL_h = allowable_decay_mV / life_damage_rate_mV_h

ECSA_ratio_proxy = 1 - (1 - ECSA_EOL_ratio) * sat01(damage_index)
```

默认参数：

```text
base_decay_mV_h = 0.012 mV/h
allowable_decay_mV = 75 mV
ECSA_EOL_ratio = 0.60
```

注意：`base_decay_mV_h` 是相对比较默认值，不是本项目电堆实测寿命参数。没有长期耐久数据前，结果只用于不同方案之间的相对排序。

## 4. 为什么没有直接使用完整 Pt/ECSA 机理方程

`2.pdf`、`4.pdf`、`6.pdf` 的 Pt/ECSA 模型需要 Pt 载量、粒径分布、催化层厚度、ECSA 初值、Pt 离子扩散、氧化覆盖率、CV/AST 标定数据等。当前项目缺这些参数，直接耦合会让模型看起来更复杂，但定量可信度更低。

所以第一版保留 `ECSA_ratio_proxy` 作为慢变量接口。后续如果拿到 ECSA 或长期电压衰减数据，可以把 `life_damage_rate_mV_h` 替换或修正为 Pt/ECSA 半机理方程。
