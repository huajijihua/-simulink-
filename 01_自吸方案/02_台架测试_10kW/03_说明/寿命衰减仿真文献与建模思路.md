# 寿命衰减仿真文献与建模思路 v01

## 1. 建模目标

当前 cEGR 仿真的寿命拓展目标不是建立完整的 PEMFC 老化数字孪生，而是在已有 10 kW 电堆/台架模型上增加一个可解释、可标定、可对比的寿命衰减计算模块，用于回答：

1. 急速/低负荷工况下，原状态与 cEGR 最佳结果的寿命衰减差异。
2. 所有工况点在进气湿度提升后的寿命变化。
3. cEGR 降低怠速高电位和提升入口湿度叠加后的寿命改善趋势。

第一版模块应输出相对寿命收益，不应直接宣称绝对寿命小时数。原因是当前缺少本项目电堆长期耐久实验、ECSA 测试、Pt 载量/粒径分布等材料参数。

## 3. 文献可直接提取的建模结论

### 3.1 高电位是低负荷寿命衰减的核心触发项

低电流、怠速或近开路状态会造成单片电压偏高，促进 Pt 催化剂溶解、碳腐蚀和材料衰减。`5.pdf` 明确指出，高电位状态会导致 Pt 溶解和碳材料腐蚀，0.65 V 到 OCV 的电位循环衰减率可显著高于 0.65 V 到 0.85 V 的循环。`3.pdf` 也指出低负荷/怠速下高电位和低产水会造成关键材料腐蚀和衰减。

对本项目的直接含义：寿命模块必须把 `V_cell_sim` 作为核心输入，尤其关注 `V_cell_sim > 0.8 V` 或超过设定阈值的暴露量。

### 3.2 cEGR 的寿命收益链条

cEGR 对寿命的作用不是直接作用在老化方程上，而是通过改变入口组分和水状态：

```text
EGR率提升
  -> 阴极入口氧摩尔分数下降 / 氧分压下降
  -> 低负荷单片电压下降
  -> 高电位暴露量下降
  -> Pt/ECSA相关衰减降低

EGR率提升
  -> 回流湿气提升阴极入口RH
  -> 膜水状态改善
  -> 欧姆损失、膜干燥风险和水管理相关衰减改善
```

`5.pdf` 的 10 kW 实验表明，阴极循环和降低新鲜空气量可以明显降低低电流单片电压，并且阴极循环具有自加湿效果。`3.pdf` 进一步把高电位控制目标转化为氧分压调节问题。该逻辑与本项目已有输出字段 `xO2_ca_in`、`pO2_ca_in_kPa`、`RH_ca_in`、`V_cell_sim` 完全匹配。

### 3.3 经验寿命模型最适合第一版 Simulink 接入

`1.pdf` 提出的负载谱经验模型最适合本项目第一版。核心做法：

1. 对电流密度序列 `i(t)` 做离散傅里叶变换，得到电流频谱，计算电流负载特征值 `m_cur`。
2. 对单片电压序列 `v(t)` 做直方图统计，按高/低风险电压区间加权，计算电压负载特征值 `m_vol`。
3. 负载特征值 `phi = m_cur * m_vol`。
4. 电压衰减率近似为 `v_deg_rate = a * phi + b`。
5. 寿命可按允许电压衰减量除以衰减率估算；文献使用 `0.1 * V_rated` 作为允许衰减电压。

文献推荐较合适的电压运行区间为约 `0.6-0.8 V`。因此本项目第一版可使用 `0.8 V` 作为高电位惩罚阈值，与仿真要求中“单片电压 0.8 V 上限”一致。

### 3.4 Pt/ECSA 机理模型适合第二版，不适合直接作为第一版主模型

`2.pdf`、`4.pdf`、`6.pdf` 都指向 Pt 催化剂退化机理：Pt 溶解、Pt 氧化、Pt 离子扩散、Ostwald ripening、粒径分布变化和 ECSA 损失。它们能解释寿命衰减本质，但直接落地需要：

- Pt 载量、Pt/C 比、粒径分布、催化层厚度、离聚物体积分数。
- ECSA 初值和耐久测试校准数据。
- 电位循环、温度、RH、pH、Pt 离子扩散等参数。
- 刚性 ODE/PDE 数值处理。

这些参数当前项目不完整。因此第二版可以实现一个降阶 `ECSA_loss` 状态，但第一版应以经验电压衰减率和高电位暴露量为主。

## 4. 第一版寿命模块建议

### 4.1 模块名称

建议新增模块命名为：

```text
LifeDegradationEstimator
```

实现位置建议：

```text
CEGR_TestBench_10kW_v01.slx
  -> StackAndVoltage / LifeDegradationEstimator
```

若先不改 Simulink，可先在 MATLAB 后处理脚本中实现同名函数，对现有 CSV 结果进行寿命指标计算。

### 4.2 输入输出接口

| 接口 | 单位 | 来源 | 说明 |
|---|---:|---|---|
| `time_s` | s | 仿真时间 | 用于积分暴露量和负载谱窗口。 |
| `current_density_A_cm2` | A/cm2 | 已有电流/面积 | 寿命模型主负载输入。 |
| `V_cell_sim` | V | 电压模型 | 高电位暴露和电压直方图输入。 |
| `egr_fraction_cmd` | 1 | EGR 控制 | 用于结果分组和寿命收益解释，不建议直接作为衰减因子。 |
| `pO2_ca_in_kPa` | kPa | 阴极入口状态 | 氧分压调节和高电位原因解释。 |
| `RH_ca_in` | 1 | 阴极入口状态 | 湿度衰减修正项或风险指标。 |
| `T_stack_C` | degC | 热模型 | 温度加速因子输入，第一版可先置为 1 或弱修正。 |
| `normal_operation_ok` | bool | 风险判据 | 非正常点不参与正式寿命收益排序，只做风险提示。 |

| 输出 | 单位 | 说明 |
|---|---:|---|
| `high_potential_exposure_Vs` | V*s | `max(V_cell - V_safe, 0)` 的时间积分。 |
| `voltage_damage_factor` | 1 | 电压直方图或稳态电压惩罚因子。 |
| `humidity_damage_factor` | 1 | 湿度风险修正，第一版可选。 |
| `life_damage_rate` | mV/h 或 1/h | 相对衰减速率。 |
| `delta_V_deg_mV` | mV | 累计电压衰减量。 |
| `life_benefit_ratio` | % | 相对无 EGR 基准的衰减降低比例。 |

### 4.3 第一版核心公式

稳态工况扫描优先使用简化形式：

```text
V_safe = 0.8
V_low = 0.6

f_V = 1 + k_high * max(V_cell - V_safe, 0)^2
          + k_low  * max(V_low - V_cell, 0)^2

f_RH = 1 + k_dry * max(RH_min - RH_ca_in, 0)^2

damage_rate = damage_rate_base * f_V * f_RH * f_T

delta_V_deg_mV = damage_rate * duration_h
life_benefit_ratio = 1 - damage_rate_EGR / damage_rate_noEGR
```

动态负载循环再扩展为 `1.pdf` 的负载谱形式：

```text
m_cur = weighted_spectrum(i(t))
m_vol = weighted_histogram(V_cell(t))
phi = m_cur * m_vol
v_deg_rate = a * phi + b
```

对当前项目而言，第一阶段先用稳态简化公式更稳妥。原因是当前仿真任务主要是 EGR 率、恒电流、恒电压、进气湿度等稳态或准稳态对比，不是完整车辆动态循环。


其中可直接使用：

```text

egr_fraction_cmd
V_cell_sim
pO2_ca_in_kPa
RH_ca_in
T_stack_C
normal_operation_ok
risk_label
```

示例口径：

1. `3-a` 急速/低负荷原状态 vs 最佳结果：选择同一电流密度下无 EGR 和最佳 EGR 点，比较 `V_cell_sim`、`high_potential_exposure`、`damage_rate`。
2. `3-b` 进气湿度提升：在所有工况点上调整 `RH_ca_in` 或加载湿度提升工况结果，比较 `f_RH` 和总 `damage_rate`。
3. `3-c` 叠加计算：使用最佳 EGR 点和湿度提升后的 `RH_ca_in`，计算组合后的 `damage_rate`，与无 EGR 基准比较。

注意：如果某个点 `normal_operation_ok=0` 或 `risk_label` 显示氧分压目标未达、DQ60 边界、振荡等问题，应标为“趋势参考”，不应作为正式最优寿命结论。

## 6. 建议实施顺序

### 阶段 A：后处理寿命指标，不改 Simulink

1. 新增 MATLAB 后处理脚本，读取已有 CSV。
2. 计算每个工况的 `f_V`、`f_RH`、`damage_rate`、`life_benefit_ratio`。
3. 输出寿命对比 CSV 和汇总图。
4. 在汇报图中展示“EGR率-单片电压-寿命衰减率”和“EGR率-进气湿度-寿命衰减率”。

这个阶段最快，适合先满足仿真要求第 3 项。

### 阶段 B：Simulink 中加入寿命估算模块

1. 在电堆电压模块后增加 `LifeDegradationEstimator`。
2. 使用 Integrator 或 Discrete-Time Integrator 积分 `delta_V_deg_mV`。
3. 输出寿命状态到 logsout 或 To Workspace。
4. 保持寿命模块不反向影响电压模型，避免第一版引入新的标定不确定性。

### 阶段 C：半机理 ECSA 状态

1. 增加归一化 ECSA 状态 `ECSA_ratio`。
2. 用高电位、电位循环幅值、温度、RH 计算 `dECSA/dt`。
3. 将 `ECSA_ratio` 影响 activation loss，例如交换电流密度或活化损失修正。
4. 需要 ECSA 或长期衰减实验后再做定量标定。

## 7. 结论

当前最合理路线是：第一版使用 `1.pdf` 的负载谱/电压衰减经验模型思想，并针对本项目稳态工况简化成高电位暴露和湿度修正模型；用 `3.pdf`、`5.pdf` 和已有 cEGR 文献支撑“阴极循环降低低负荷电压、提高入口湿度、降低寿命衰减”的因果链；用 `2.pdf`、`4.pdf`、`6.pdf` 作为后续 ECSA/Pt 机理模型参考。

因此本项目寿命模块第一版应定位为“相对寿命收益估算器”，可以用于方案对比和汇报展示；若要给绝对寿命小时数，需要补充长期耐久实验或至少给出本项目电堆的电压衰减率标定点。
