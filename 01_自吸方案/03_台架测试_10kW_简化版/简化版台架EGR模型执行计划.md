# 简化版台架EGR模型执行计划

## 任务边界

本任务在 `01_自吸方案\03_台架测试_10kW_简化版` 内建立新的简化版台架测试结构模型。该工程包与 `02_台架测试_10kW` 的完整台架模型、寿命模型、条件扫描脚本隔离，不沿用旧的研究脚本和验证图片。

## 建模原则

1. 以 Simulink 模型为主，脚本仅作为初始化、数据整理、参数拟合和 MATLAB 图窗展示的辅助。
2. 新建干净的简化版模型，复用已有完整电堆核心，不简化电堆内部阴阳极流道、电压、热、膜水、液态水、相变、歧管压力-流量关系和源项表达式。
3. 简化对象仅限台架/BOP外围：忽略空压机和中冷器物理模型；水分离器只去除液态水；阀门、三通只承担流量分配/经验映射功能。
4. 第一版按稳态工况点模型处理，不建立动态台架控制器。
5. 不生成 PNG 等图片文件；仿真结果展示使用 MATLAB 绘图窗口。

## 数据口径

1. 无EGR基础极化曲线来自 `10kw电堆台架稳态测试数据整理.xlsx`，并加入补充点 `20A / 0.05 A/cm2 / EGR率=0 / 单片电压=0.818V`。
2. EGR数据来自 `10kw短堆阴极尾气循环测试_0.1Acm2加20A补充_整理版_修正.xlsx`。
3. 循环比主口径优先级：
   - `回流参数_校准EGR率`
   - `回流参数_校准回流 / 回流参数_入堆流量流量计(SLPM)`
   - `(回流参数_入堆流量流量计(SLPM) - 台架供给_给气流量(SLPM)) / 回流参数_入堆流量流量计(SLPM)`
4. 所有流量读入后统一换算为模型内部单位，避免体积流量、质量流量、摩尔流量混用。

## 模型结构

1. 台架供给边界：按测试点给定电堆入口总流量、温度、压力、湿度和新鲜空气组分。
2. EGR回流边界：按循环比从电堆阴极出口气中取回流气体，并与台架新鲜供给混合。
3. 水分离器：去除液态水，保留水蒸气和干气组分；温度压力由传感器数据或经验插值外部给定。
4. 阳极侧：保留为台架边界输入，不做阳极循环；EGR测试缺失阳极数据时由无EGR数据按电流插值。
5. 电堆核心：复用已有模型中完整电堆核心，外部接口适配简化台架边界。

## 参数拟合策略

1. 无EGR完整极化曲线用于基础电堆参数拟合。
2. EGR稳态点只用于EGR敏感参数、边界修正参数和经验插值参数拟合。
3. Nernst形式、法拉第耗氧产水关系、质量守恒、能量守恒和组分混合关系不作为自由拟合项。

## 验收指标

1. 无EGR极化曲线：单片电压误差、低中高电流段误差、边界输入一致性。
2. EGR趋势：0.1 A/cm2下循环比升高时电压下降趋势可复现；0.05 A/cm2补充点作为弱验证。
3. 入口组分：入堆氧摩尔分数随循环比下降，实际氧计量比由混合后氧流量计算。
4. 守恒检查：氧消耗、氮守恒、水生成/去除/回流闭合、循环比和单位换算链一致。

## 执行步骤

1. 检查已有车载/台架模型中可复用电堆核心的接口。
2. 生成最小数据整理和初始化脚本。
3. 新建简化版 Simulink 模型，建立台架供给、EGR混合、水分离、阳极边界、电堆核心和诊断输出。
4. 运行模型结构检查和初始化脚本检查。
5. 如模型可运行，进入基础标定；如接口不匹配，先修正边界适配，不改电堆内部方程。
