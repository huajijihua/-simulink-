# 简化版台架 EGR 模型执行计划

## 当前目标

本目录只保留一个干净的 10 kW 台架简化 EGR 模型，用于回放已整理的无 EGR 和 CEGR 稳态测试点，检查气体/水/热/电压链路是否能支撑后续研发判断。

模型当前边界是稳态工况回放，不是台架控制器、空压机模型或通用条件扫描模型。

## 核心文件

- 模型：`01_模型/CEGR_TestBench_10kW_SimplifiedEGR_v01.slx`
- 初始化：`02_脚本/init_testbench_10kw_simplified_egr.m`
- 标定：`02_脚本/calibrate_testbench_10kw_simplified_egr.m`
- 审计：`02_脚本/run_core_fix_v01_audit.m`
- 合并工况数据：`00_输入参数/实验数据/combined_noegr_cegr_fit_points.csv`
- 标定参数：`00_输入参数/标定参数/simplified_noegr_stack_params.csv`

## 建模口径

1. 台架简化模型采用实测入堆总质量流量作为阴极入口质量流量边界。
2. 阴极入口压力、阴极出口压力和阴极压差作为诊断和验证量保留；入口不维护压力-流量系数。
3. 阴极出口仍使用压力-流量关系，出口背压只来自活动表中的 `stack_out_p_kPa`。缺失出口压力必须先在活动表中按相似工况补齐，不在初始化脚本里运行时兜底。
4. 电堆出口节点是 EGR 分流前的堆出口，出口流量和出口组分由电堆模型计算，不从实验表直接指定。
5. CEGR 回流按循环比从阴极出口气体分流，经水分离器去除液态水后回到混合器；分离器节点只读取回流温度 `egr_return_T_C` 和回流压力 `egr_return_p_kPa`。
6. 回流 RH 只作为实验记录/校核量，不进入模型参数；分离器湿度和冷凝由回流气体组分、温度和压力计算。
7. 无 EGR 工况不经过分离器；脚本保留分离器温压只是为了统一初始化，不作为有效气路边界。
8. 阳极侧没有循环模型，缺失 CEGR 阳极边界时按无 EGR 数据随电流插值。
9. 电压影响只能通过 `pO2/RH/T/lambda_m/etaOhm/etaCon` 等物理链路体现，不使用 EGR 比例直接扣电压。

## 单位规则

模型内部压力使用 `kPa`，出口压力-流量系数变量名统一为 `kg_s_kPa`。

书籍或文献若给出 `kg/(Pa*s)`：

```text
K_kg_s_kPa = 1000 * K_kg_s_Pa
```

因此 `1e-9~1e-8 kg/(Pa*s)` 对应模型中的 `1e-6~1e-5 kg/(kPa*s)`。

当前台架入口按实验入堆流量直接给定，不设置、不维护入口流量系数。文献量级只用于检查出口系数或未来阀口模型，不能回写成入口截断逻辑。

出口系数仍参与阴极库存闭合，但不能简单把文献 `1e-9~1e-8 kg/(Pa*s)` 硬套进当前零维等效体积。临时筛查显示，`1e-6~1e-5 kg/(kPa*s)` 可运行但会导致内部阴极压力相对出口背压异常升高；出口边界需要单独重定，而不是混在当前入口边界清理中。

## 求解口径

1. `P.dt_s`、`StackParam(35)`、模型 FixedStep 和两个 UnitDelay 的 SampleTime 必须一致。
2. 当前核心仍是 MATLAB Function 内部显式离散更新加 UnitDelay 回灌，不是严格连续 ODE。
3. 膜水通量使用一阶松弛状态 `mMem_state`，`StackParam(79)=tau_mem_s`。
4. 后续若升级严格 ODE，应作为单独结构任务处理，不与电压标定混在一起。

## 验证判据

每次核心模型或边界脚本改动后，至少运行：

```matlab
run_core_fix_v01_audit(120)
```

通过判据：

- 29 个工况全部成功，无 NaN/Inf。
- `mIn_kg_s` 末端不为 0，且与实测入堆质量流量一致。
- `lambdaO2` 由混合后真实氧流量计算，不能出现由入口截断导致的伪 0。
- 氧/氮/水/氢质量残差维持在数值误差量级。
- 入口氧摩尔分数随 EGR 升高整体下降。
- 电压误差作为后续标定指标，不作为当前边界正确性的唯一判据。
